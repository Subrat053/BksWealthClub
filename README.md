# BksWealthClub Monorepo Scaffold

## Apps
- `apps/user-web`: Public website + member panel (React + Vite + Tailwind)
- `apps/admin-web`: Admin dashboard (React + Vite + Tailwind)
- `server`: Node + Express backend scaffold

## Quick Start
1. Install root dependencies: `npm install`
2. Run user app: `npm run dev:user`
3. Run admin app: `npm run dev:admin`
4. Run backend: `npm run dev:server`
